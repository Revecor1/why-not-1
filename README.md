# Почему Нет? — кофейня (дипломный проект)

Веб-сайт кофейни в Ульяновске: Django REST API + статический фронтенд. Сайт, API и админ-панель работают **с одного сервера** на порту 8000.

**Репозиторий:** https://github.com/Revecor1/why-not-1

## Возможности

- **Меню** — 28 позиций в 5 категориях, корзина и оформление заказа
- **Бронирование столиков** — с валидацией даты, времени и телефона
- **Новости и акции** — публичная страница + управление через админ-панель
- **Личный кабинет** — история заказов и бронирований
- **Чат поддержки** — переписка пользователя с администратором
- **Админ-панель** — заказы, брони, чаты, новости
- **Карта** — точное расположение на Яндекс Картах (ул. Карла Маркса, 18)

## Требования

- **Windows 10/11**
- **Python 3.10+** — [python.org](https://www.python.org/downloads/)
  - При установке включите **«Add Python to PATH»**

## Перенос на другой компьютер

1. Склонируйте репозиторий или скопируйте папку проекта.
2. **Не копируйте** папку `venv` — она создаётся заново на каждом ПК.
3. На новом компьютере запустите **`setup.bat`** (один раз).
4. Для работы используйте **`start.bat`**.

## Ручной запуск

```cmd
cd путь\к\проекту
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
python manage.py migrate
python seed_data.py
python manage.py runserver 127.0.0.1:8000
```

## API

Базовый URL: `http://127.0.0.1:8000/api/`

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/menu/` | Меню |
| GET/POST | `/orders/` | Заказы |
| POST | `/orders/{id}/complete/` | Отметить заказ выполненным (админ) |
| GET/POST/DELETE | `/reservations/` | Бронирования |
| GET/POST | `/chats/` | Чат |
| GET | `/news/` | Опубликованные новости |
| POST/PATCH/DELETE | `/news/` | Управление новостями (админ) |
| POST | `/auth/login/` | Вход |
| POST | `/auth/register/` | Регистрация |
| POST | `/auth/logout/` | Выход |
| GET | `/auth/me/` | Текущий пользователь |

## Структура проекта

| Папка / файл | Назначение |
|--------------|------------|
| `api/` | Модели, сериализаторы, API, тесты |
| `coffee_backend/` | Настройки Django, раздача фронтенда |

## Важно

- **Не открывайте** HTML через `file://` — авторизация, корзина и API не будут работать.
- Используйте только **`start.bat`** или `runserver` — сайт и API должны быть на одном адресе.
- Часовой пояс бронирования: **Europe/Samara** (Ульяновск).
- Папки `venv/` и `db.sqlite3` не хранятся в git — создаются локально при установке.
