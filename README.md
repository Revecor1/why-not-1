# ☕ Почему Нет? — Кофейня

Веб-приложение для кофейни с меню, корзиной, бронированием столиков и чатом поддержки.

## Стек

- **Backend:** Python, Django 5.2, Django REST Framework
- **Frontend:** HTML, CSS (Tailwind), JavaScript
- **База данных:** SQLite

## Запуск

**1. Установить зависимости**
```bash
cd "Why not"
pip install -r requirements.txt
```

**2. Создать базу данных**
```bash
python manage.py migrate
```

**3. Заполнить меню**
```bash
python seed_data.py
```

**4. Запустить Django**
```bash
python manage.py runserver
```

**5. Запустить сервер для фронтенда** (в отдельном терминале)
```bash
python -m http.server 3000
```

**6. Открыть в браузере:** `http://127.0.0.1:3000/index.html`

- Просмотр меню с фильтрацией по категориям

- Бронирование столиков
- Личный кабинет с историей заказов
- Чат поддержки
- Панель администратора
